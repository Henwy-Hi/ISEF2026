# Effective Upcycling of PET Plastics

This project focused on solving an issue in the plastics industry where almost all new plastic produced will end up in landfills, even if recycled. If plastic could be broken down into it's raw components in a cost-effective way, we can remove crude oil from the production demand and use plastic waste in the production of new plastics. This would create a circular free market economy where the most profitable alternative is the most environmentally friendly.
